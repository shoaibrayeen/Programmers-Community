# An Array is given and every element is appearing twice except one element. You need to find that element.

# Constraints

- 1 < size Of Array <= 10^6
- 0 < Array Elements <= 10^9

# Input

- Array
- Its Size

# Output

- Element that occurs only once.

```
Input :  [1, 2, 3, 1, 4 , 3 , 4]
Output : 2 

Input :  [1 , 3, 2, 3, 1]
Output : 2
```
